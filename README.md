# Interaction Structure and Feedback Asymmetry Leading to Heterogeneity in Robot Swarms

All files including the simulation data can also be found on this google drive, since the data were too large to be uploaded to GitHub, even as a zip file.
Link to the google drive: https://drive.google.com/file/d/1JGmCx_SQBjJOdPT5KHG3wIoP_s7zvEsP/view?usp=sharing 

This repository includes: 

- experiment.py --> the code to run a single simulation
- run_experiments.py --> the code to set the interaction weights and run a set of experiments
- graphics --> folder with all graphics needed to run a simulation

Additionally, the google drive includes:

- simulation_data.zip --> zip file with the data of 3 swarm sizes x 30 seeds of simulations with uniform interaction weights
- baseline_simulation_data.zip --> zip file with simulation data of 3 swarm sizes x 30 seeds without using interaction weights
- final_results_uniform.zip --> pipeline output for the simulations in simulation_data.zip, including their meta role analysis

To run the code, make sure to have an up-to-date version of python as well as the necessary libraries installed.
